# practica-colaborativa
# Nombre del grupo: [Sabor IA ¿Qué comeremos hoy?]
## Curso: Herramientas de Desarrollo
## Descripcion: [Aplicación web colaborativa que recomienda platos o comidas según el presupuesto del usuario, los ingredientes disponibles y el momento del día. El sistema permite ingresar insumos como pollo, arroz, verduras o un monto de dinero, y sugiere opciones de comidas prácticas, incluyendo platos locales peruanos. Además, podrá gestionar recetas, historial de búsquedas y recomendaciones personalizadas, utilizando herramientas de control de versiones, trabajo colaborativo, despliegue en la nube y contenedores, alineado al curso de Herramientas de Desarrollo.]
